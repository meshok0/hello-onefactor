# hello-onefactor
Test for OneFactor

All steps are tested on Centos 7 x64


# To build jar:
```
% sudo yum install git maven rpm-build -y
```
make java7 as a default:
```
% sudo alternatives --config java
```
```
% git clone https://github.com/meshok0/hello-onefactor.git
% cd hello-onefactor/
% mvn package
```

To build rpm:
```
% cd ~

```
